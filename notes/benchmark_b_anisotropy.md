# Benchmark B: Anisotropy Steering

## Purpose

Measure whether the anisotropy-aware term can steer interface orientation without destroying geometric alignment.

## Reported quantities

- gradient component ratio along the prescribed stress direction
- alignment RMS against the geometric target
- optional path-heading changes in the warped G-code

## Expected outcome

Moderate steering weights should noticeably change orientation behavior while causing only marginal loss in alignment quality.
