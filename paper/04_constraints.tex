\section{Manufacturability Constraints}

The representation is designed so that manufacturability arguments remain explicit.

\subsection{Slope inheritance}

Because `w` depends only on `z_s`, the in-plane gradient is

\[
\nabla_{xy} D = (1-w)\nabla S_{\text{low}} + w \nabla S_{\text{high}}.
\]

There is no `\nabla w` term. The gradient is therefore a convex combination of anchor gradients, which means a shared slope bound on the anchors propagates naturally to the full field.

\subsection{Thickness and invertibility}

Vertical scaling depends on

\[
\frac{\partial D}{\partial z_s} = w'(z_s)\left(S_{\text{high}} - S_{\text{low}}\right).
\]

This term controls both effective layer thickness and the invertibility condition `1 + \partial D / \partial z_s > 0`. The blend interval must therefore widen as anchor separation grows.
